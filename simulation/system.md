一、 核心概念与方法框架
该方法的核心是接触点位移变化（Contact Point Displacement Variation, CPDV）。CPDV是通过计算车辆过桥时，损伤状态与完好状态下的表观剖面（Apparent Profile, AP） 之差得到的。AP是车辆-桥梁接触点的位移时程与路面不平度之和。
整体损伤检测算法流程如下：

数据生成：通过有限元模拟，设置桥梁不同位置和程度的单/多损伤，获取车辆加速度时程。
CPDV计算：利用车辆加速度时程和车辆动力学模型，反算AP，并通过与完好状态AP相减得到CPDV。
神经网络训练与预测：以CPDV作为输入，损伤参数（位置、程度）作为输出，训练BP神经网络，建立映射关系。
验证：通过改变车速、桥梁阻尼比等参数测试网络鲁棒性，并通过缩尺模型实验验证。

二、 求解接触点位移（CPDV）的具体数值方法
求解过程涉及两个主要部分：车辆-桥梁相互作用模拟和AP/CPDV反算。
第一部分：车辆-桥梁相互作用模拟（获取车辆加速度）
采用非耦合迭代程序来数值求解车辆加速度时程。将车辆过桥全过程分为多个时间步，在每个时间步独立求解车辆和桥梁两个子系统的运动方程。
1. 基本方程：
车辆与桥梁的运动方程（对于1/4车模型）在时间步 i 表示为：

桥梁子系统： [M_b]{a_b}_i + [K_b]{w_b}_i + [C_b]{v_b}_i = {f_b}_i
车辆子系统： [M_v]{a_v}_i + [K_v]{w_v}_i + [C_v]{v_v}_i = {f_v}_i
其中，[M], [K], [C] 为质量、刚度、阻尼矩阵，{f} 为力向量，w, v, a 为位移、速度、加速度向量。

2. 迭代求解步骤（6步算法）：

步骤1：初始化。假设每个时间步接触点（CP）位移为零。
步骤2：计算车辆轴上的接触力。公式：f_va_i = k_a * (w_bi + r_ci)，其中 k_a 为车轴刚度，w_bi 为当前时间步桥梁在接触点的挠度，r_ci 为当前时间步接触点的路面不平度。
步骤3：求解车辆加速度。将步骤2得到的接触力作为车辆方程右端的力向量，使用 Newmark-β法 独立求解每个时间步的车辆加速度 {a_v}_i。
步骤4：计算作用于桥梁的全局力向量。该力由车辆重力（静载）和车辆惯性力（动载）组成。通过接触点所在单元的形函数向量 {N_b}_i 和单元定位向量 [L_i]，将接触点上的外力转换到桥梁有限元模型的所有节点上，形成桥梁的全局力向量 {f_b}_i。
步骤5：求解桥梁全局位移。将步骤4得到的全局力向量作为桥梁方程右端的力向量，使用 Newmark-β法 求解每个时间步的桥梁全局位移向量 {w_b}_i。需在此步考虑桥梁边界条件。
步骤6：收敛性判断。收敛准则为：桥梁质心处最大位移在本轮迭代计算值与上一轮计算值的相对误差小于 1%。若未收敛，则用步骤5得到的桥梁全局位移向量更新步骤2中的接触点位移 w_bi，开始新一轮迭代（返回步骤2）；若收敛，则迭代结束，获得最终的车辆加速度时程。

第二部分：从车辆加速度反算AP与CPDV
在实际“驾驶检测”中，直接测量接触点位移时程困难，但测量车辆加速度时程可行。因此，需从测量的车辆加速度反算AP。
1. 修改车辆模型：
在原有车辆模型基础上，增加一个质量近乎为零的接触点自由度，专门用于计算接触点位移时程（即AP）。修改后的车辆动力学方程为：
[M_v]‘{a_v}_i + [K_v]’{u_v}_i + [C_v]’{v_v}_i = {f_r}_i
其中，带 ‘ 的矩阵为考虑了接触点自由度后的修正质量、刚度、阻尼矩阵。{f_r}_i 为作用于接触点的惯性力，计算公式为：f_r_i = -{a_v}_i^T [M_v] {1; 1}。
2. 求解AP与CPDV：

使用 Newmark-β法 求解上述修正后的车辆方程，即可得到 AP（Apparent Profile），即接触点位移时程 u_c。
CPDV 通过下式计算：CPDV = AP_damaged - AP_intact，即损伤状态下的AP减去完好状态（基线）下的AP。

三、 关键数值问题与处理

迭代收敛性：整个车辆-桥梁相互作用模拟是一个迭代过程，收敛性是获得准确解的关键。采用桥梁质心位移的相对误差（<1%）作为收敛判据。
时间积分方法：在求解车辆和桥梁的动力方程时，均采用了 Newmark-β法 进行直接积分。
数据预处理：在将CPDV数据输入BP神经网络前，进行了归一化处理（最小-最大缩放至[0,1]区间），以消除样本数据的奇异性，提高网络训练效率和泛化能力。
阻尼处理：桥梁阻尼采用经典的粘性瑞利阻尼模型 [C_b] = α[M_b] + β[K_b]。研究表明，桥梁阻尼比增大虽会降低车辆加速度频谱峰值，使接触点位移减小，增加损伤检测难度，但对所提方法（基于CPDV和BPNN）的检测精度影响很小。
边界条件：在组装桥梁整体刚度、质量、阻尼矩阵后，必须施加边界条件。

对于一维简支梁：两端节点的竖向位移约束为零。
对于二维对边简支板：简支边上每个节点的竖向位移和垂直于简支边的转角约束为零。

四、 实验验证
通过铝板缩尺模型实验验证了该数值方法的有效性。在桥梁底部添加质量块模拟损伤，车辆以固定速度和路线过桥，采集车辆加速度数据。利用车辆动力学模型和传感器采集的加速度计算CPDV，并使用一维卷积神经网络进行损伤识别。实验结果（R², MSE）表明，该基于CPDV的框架在实际应用场景中具有良好的预测性能，满足工程应用精度要求。
总结：求解接触点位移变化（CPDV）的核心是一个包含迭代的数值模拟-反算流程。首先通过车辆-桥梁耦合迭代有限元模拟获得车辆加速度，然后利用修正的车辆模型和Newmark-β法从加速度反算表观剖面（AP），最后通过差分得到CPDV。整个过程涉及收敛性控制、时间积分、数据归一化等多个数值处理环节。
